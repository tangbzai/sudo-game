# sudo-game

> 一个基于 Vite 和 React 的数独游戏应用。

## 📝 简介

这是一个使用 [Vite](https://vitejs.dev/) 构建工具和 [React](https://reactjs.org/)
框架开发的数独（Sudoku）游戏应用。

该项目采用模块化结构、TypeScript 编写，并集成了现代前端开发的最佳实践。

## 🚀 [在线游玩](https://tangbzai.github.io/sudo-game/)

## 🛠️ 本地运行

### 克隆项目

```bash
git clone https://github.com/tangbzai/sudo-game.git
cd sudo-vite-react
```

### 安装依赖

```bash
pnpm install
```

> ⚠️ 本项目使用 [pnpm](https://pnpm.io/) 作为包管理器，请确保你已安装 `pnpm`。

### 启动开发服务器

```bash
pnpm run dev
```

访问 `http://localhost:5173` 查看运行中的应用。

### 构建生产版本

```bash
pnpm run build
```

构建后的文件位于 `dist/` 目录。


## 🧰 技术栈

- [Vite](https://vitejs.dev/) - 快速的前端构建工具
- [React](https://reactjs.org/) - UI 组件框架
- [TypeScript](https://www.typescriptlang.org/) - 类型安全语言
- [CSS Modules](https://github.com/css-modules/css-modules) - 样式模块化支持
- [PostCSS](https://postcss.org/) - CSS 工具链，用于自动前缀和响应式适配
